## 🚀 Быстрая настройка

### 1. Добавьте бота в администраторы канала

**Обязательно!** Бот должен быть администратором канала для проверки подписки.

1. Откройте ваш канал новостей (`@vpvks_news`)
2. Зайдите в **Управление каналом** → **Администраторы**
3. Нажмите **Добавить администратора**
4. Найдите вашего бота по username (например, `@VPVKS_Bot`)
5. Дайте права:
   - ✅ **Просмотр сообщений** (обязательно)
   - ❌ Остальные права не нужны
6. Сохраните

---

### 2. Настройте переменные окружения

Добавьте в `.env` (на сервере и локально):

```bash
# Telegram каналы (для проверки подписки)
CHANNEL_NEWS_URL=https://t.me/vpvks_news
CHANNEL_NEWS_ID=vpvks_news  # ID канала для проверки подписки (без @)
CHANNEL_WIN_MAC_URL=https://t.me/vpvkspc
CHANNEL_ANDROID_IOS_URL=https://t.me/VPVKSinstr
```

**Важно:** `CHANNEL_NEWS_ID` — это ID канала **без** `@`. Например:
- `vpvks_news` для `@vpvks_news`
- `channel_name` для `@channel_name`

---

### 3. Перезапустите бота

```bash
# На сервере
docker compose restart vpn_bot

# Проверка логов
docker compose logs vpn_bot -f
```

В логах должно быть:
```
✅ JobQueue настроен: уведомления об истечении в 10:00
✅ JobQueue настроен: синхронизация Marzban каждые 5 минут
```

## 📱 Как это работает

### Для новых пользователей:

1. Пользователь нажимает `/start`
2. Бот проверяет подписку на `@vpvks_news`
3. **Если не подписан:**
   ```
   ⚠️ Для доступа к VPN приложению необходимо подписаться на канал новостей!
   
   1️⃣ Нажмите кнопку «📢 Подписаться на канал»
   2️⃣ Подпишитесь на канал
   3️⃣ Вернитесь в бот и нажмите «✅ Я подписался»
   ```
4. После подписки — открывается доступ к Mini App

### Для подписанных пользователей:

```
👋 Привет, @username!

✅ Вы подписаны на канал новостей

[🚀 Открыть VPN приложение]
[📚 Инструкции]
[📰 Новости VPVKS]
```

## 🔧 Технические детали

### Файлы:

| Файл | Описание |
|------|----------|
| `bot/config.py` | Конфигурация каналов |
| `bot/main.py` | Основная логика бота |
| `bot/utils/subscription_checker.py` | Проверка подписки |

### API Telegram:

```python
# Проверка подписки
member = await bot.get_chat_member(chat_id=CHANNEL_NEWS_ID, user_id=user_id)
is_subscribed = member.status in ['creator', 'administrator', 'member']
```

### Обработка исключений:

- Если бот **не администратор** в канале → ошибка при проверке
- Если канал **приватный** → нужен ID канала (например, `-1001234567890`)
- Если ошибка → доступ **не блокируется** (защита от ложных срабатываний)

## 🛠️ Troubleshooting

### Бот не проверяет подписку

**Проблема:** Бот не добавлен в администраторы канала

**Решение:**
1. Добавьте бота в администраторы канала
2. Дайте право «Просмотр сообщений»
3. Перезапустите бота

### Ошибка «Chat not found»

**Проблема:** Неверный `CHANNEL_NEWS_ID`

**Решение:**
1. Проверьте ID канала в `.env`
2. Для публичных каналов: `vpvks_news` (без `@`)
3. Для приватных каналов: `-1001234567890` (с цифрами)

### Пользователь подписался, но бот не видит

**Проблема:** Кэш Telegram API

**Решение:**
1. Подождите 1-2 минуты
2. Нажмите «✅ Я подписался» ещё раз
3. Если не помогло — перезапустите бота
