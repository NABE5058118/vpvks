# 🔧 Настройка проверки подписки — СРОЧНО

## ❗ ПРОБЛЕМА

Пользователь подписался на канал, но бот пишет "Вы всё ещё не подписаны".

## 🎯 ПРИЧИНА

**Бот не является администратором канала @vpvks_news**

Для проверки подписки Telegram требует, чтобы бот был администратором канала.

---

## ✅ РЕШЕНИЕ (5 минут)

### Шаг 1: Откройте канал

1. Откройте Telegram
2. Найдите канал **@vpvks_news**
3. Убедитесь что вы владелец/администратор

### Шаг 2: Добавьте бота в администраторы

1. В канале нажмите **Управление** (или ⚙️ Настройки)
2. Выберите **Администраторы**
3. Нажмите **Добавить администратора**
4. В поиске найдите вашего бота: **@VPVKS_Bot** (или ваш username)
5. Нажмите на бота

### Шаг 3: Настройте права

**Дайте ТОЛЬКО одно право:**
- ✅ **Просмотр сообщений** (can_post_messages)

**Остальные права НЕ НУЖНЫ:**
- ❌ Публикация сообщений
- ❌ Редактирование
- ❌ Удаление
- ❌ Бан и т.д.

### Шаг 4: Сохраните

1. Нажмите **✓** (галочка)
2. Готово!

## 🆘 Если не работает

### 1. Проверьте CHANNEL_NEWS_ID в .env

```bash
# Должно быть БЕЗ @
CHANNEL_NEWS_ID=vpvks_news
```

**НЕПРАВИЛЬНО:**
```bash
CHANNEL_NEWS_ID=@vpvks_news  # ❌ С @
```

### 2. Проверьте токен бота

```bash
# В .env
TELEGRAM_BOT_TOKEN=8321727057:AAGJJwoVRoG7wYZQPfN9-q-IM4mHA82g2cU
```

### 3. Перезапустите бота

```bash
docker compose restart vpn_bot
```

### 4. Проверьте приватность канала

Если канал **приватный**, нужен его ID:
```bash
CHANNEL_NEWS_ID=-1001234567890  # С цифрами
```
