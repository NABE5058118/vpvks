# Инструкция: Поток оплаты VPN

## 📋 Логика работы

```
/start → Приветствие → Mini App → Ключи → Тарифы → Оплата → Ключи доступны
```

---

## 🚀 Как это работает

### 1. Пользователь нажимает /start

**Бот отвечает:**
```
👋 Привет, @username!

🆔 ID: 123456789
💰 Баланс: 0 ₽

🔐 Для получения VPN ключа:
1. Откройте Mini App
2. Выберите тариф
3. Оплатите подписку
4. Получите ключ подключения

💳 Тарифы от 99₽/месяц
🚀 VLESS Reality + Trojan TLS
🌍 Сервер: Стокгольм, Швеция

Нажми кнопку ниже чтобы открыть приложение 👇
[🚀 Открыть VPN приложение]
```

### 2. Пользователь открывает Mini App

**Главный экран показывает:**
- ❌ Нет подписки / Требуется оплата
- Баланс: 0 ₽

### 3. Пользователь переходит в "Ключи"

**Автоматическая проверка:**
- Если подписки нет → переключение на "Тарифы"
- Сообщение: "Доступ к ключам доступен только после оплаты тарифа"

### 4. Пользователь выбирает тариф

| Тариф | Срок | Цена |
|-------|------|------|
| 1 месяц | 30 дней | 99₽ |
| 3 месяца | 90 дней | 299₽ |
| 6 месяцев | 180 дней | 599₽ |

### 5. Нажимает "Купить"

**Открывается YooKassa:**
- Сумма: 99₽ / 299₽ / 599₽
- Описание: "N месяцев подписки"
- Оплата картой / SBERPAY / SBP

### 6. После оплаты

**Автоматически:**
1. YooKassa отправляет webhook → `https://vpvks.ru/api/payment/webhook`
2. Backend активирует подписку
3. Mini App обновляет статус (через 3 и 10 секунд)
4. Ключи становятся доступны

### 7. Пользователь получает ключ

**Раздел "Ключи":**
- ✅ Активно
- QR-код для подключения
- Ссылка на подписку
- Инструкция по настройке

---

## 🔧 Настройка YooKassa Webhook

**Критично важно!**

1. Зайдите в https://yookassa.ru/my/settings/notifications
2. Добавьте URL: `https://vpvks.ru/api/payment/webhook`
3. Сохраните Secret Key в `.env`:

```bash
YOOKASSA_SHOP_ID=1266298
YOOKASSA_SECRET_KEY=live_...
YOOKASSA_TEST_MODE=false
YOOKASSA_RETURN_URL=https://vpvks.ru/payment-success
```

## 📊 Диагностика

### Проверка статуса подписки

```bash
# В Mini App: Главный экран → Статус
# Или через API:
curl https://vpvks.ru/api/vpn/status/699469085
```

### Проверка webhook

```bash
# Логи backend после оплаты
docker compose logs backend | grep "YooKassa webhook"
docker compose logs backend | grep "Activating subscription"
```

### Проверка платежей

```bash
# В PostgreSQL
docker exec vpn_postgres psql -U vpn_bot_user -d vpn_bot_db \
  -c "SELECT id, user_id, amount, status, created_at FROM payments ORDER BY created_at DESC LIMIT 10;"
```

## 🔍 Troubleshooting

### Ключи не открываются после оплаты

1. Проверьте webhook в логах
2. Проверьте статус платежа в БД
3. Обновите страницу Mini App

### YooKassa не возвращает на сайт

Это нормально. Пользователь должен сам вернуться в бота.
Статус обновится автоматически через 3 и 10 секунд.

### Ошибка "Подписка не активна"

Пользователь ещё не оплатил. Нужно:
1. Перейти в "Тарифы"
2. Оплатить подписку
